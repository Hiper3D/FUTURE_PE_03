# FUTURE_PE_03
# Future Interns: Task 3 - AI SEO Blog & Content Cluster Generator

## Overview
This repository contains the deliverables for Task 3 of the Prompt Engineering Internship. The objective was to design a structured prompt workflow capable of generating ranking-focused SEO content systems, including Pillar Blogs, Supporting Blogs, and keyword intent strategies.

## Business Profile 
* **Business Name:** Kalinga Cloud & IT Solutions
* **Niche:** B2B IT Services & Cloud Migration
* **Location:** Bhubaneswar, Odisha

## Prompt Engineering Logic
The `master_prompt.txt` is engineered to mirror the workflow of a professional SEO agency. Instead of simply generating an article, the prompt forces the LLM to establish the Keyword Intent first, followed by mapping out a Content Cluster (Supporting Blogs). Finally, it generates the Pillar Blog enforcing strict H1-H3 hierarchical structuring and local SEO integration (city + service parameters).

## Included Files
1. `master_prompt.txt`: The system instruction set.
2. `seo_content_pack.txt`: The AI-generated Keyword Intent Explanation, Content Cluster Map, and the H1-H3 Pillar Blog.
